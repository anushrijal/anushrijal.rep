# Futuristic Robotic AI Theme

A futuristic website theme with robotic AI-inspired design, glowing animations, and SEO-optimized features. This project is designed for easy hosting on GitHub Pages.

## Features
- Neon glowing effects and smooth animations.
- Fully responsive and mobile-friendly design.
- SEO-optimized structure with fast load times.
- Easy deployment on GitHub Pages.

## Getting Started

1. Clone this repository:
    ```bash
    git clone https://github.com/anushrijal/futuristic-theme.git
    ```

2. Open the project folder:
    ```bash
    cd futuristic-theme
    ```

3. Launch `index.html` in your browser or use a live server.

## Deployment on GitHub Pages

1. Push your changes to the repository's main branch.
2. Go to the repository's settings.
3. Under "Pages," select the branch and folder for deployment.
4. Your website will be live at `https://<your-username>.github.io/<repository-name>/`.
